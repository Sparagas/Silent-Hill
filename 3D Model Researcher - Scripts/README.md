# 3D Model Researcher — Scripts

Python scripts for **[3D Model Researcher](https://mr.game-viewer.org/)**, a tool designed for reverse engineering 3D model formats by letting you define vertex, face, and material layouts interactively.

## Scripts

| File | Game | Description |
|------|------|-------------|
| `havok_hcl.py` | Silent Hill: Homecoming | Havok collision data (.hcl) |
| `she_ios_nom.py` | Silent Hill: The Escape | iOS NOM 3D model format |
