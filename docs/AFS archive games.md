# List of Games That Use the AFS Archive Format

*Silent Hill 3* for the *PS2* and *PC*, along with *Silent Hill 4: The Room* for the *PS2*, use *CRI Middleware*'s *AFS* archive format.

Many other games also use this format, as it is a common file type adopted by numerous companies.

This list includes all known games that use *AFS* archives, but it is not exhaustive and may contain some errors. Additionally, there are multiple variations of the *AFS* format, so not all archives are identical.

Note: all file extensions are **.afs* unless noted otherwise.
