# List of Games That Use the Konami Skeletal Model Format

Silent Hill 2, Silent Hill 3, and Silent Hill 4: The Room use very similar skeletal model formats for characters, enemies, and other moving objects.  

Further inspection has revealed that many more Konami games use this format, meaning it is not exclusive to the Silent Hill series. Below is a list of identified games, along with their version information and platform details.

The earliest game found using this format is *Shadow of Destiny* (also known as *Shadow of Memories*).  
*Tokimeki Memorial 3* is the only game in the series that uses 3D graphics, and it uses this format for character models.  
*PES 1* (*Winning Eleven 5*) and *PES 2* (*Winning Eleven 6*) actually use RenderWare instead of the Konami skeletal format. Many other Konami non-*PES* football games do not appear to use this format either. Similarly, other handheld *PES* games seem to use different formats.  
While not all Xbox 360 and PS3 *PES* games were tested, it appears they also use a different format. Additionally, *PES* games for PC and Wii have not been tested yet.  
For *PS2* games, these are the only *Contra* and *Castlevania* titles that use this format.  
It also appears that the *Metal Gear Solid* and *Dance Dance Revolution* series for *PS2* do not use the Konami skeletal model format.
